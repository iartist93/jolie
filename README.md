# Jolie

Jolie is a simple component library to build more complex Vue applications

- Require Vue 2.x

## Documentation

https://iartist93.github.io/jolie/

---

## Install

```sh
> yarn add jolie
# or
> npm install jolie
```

---

## Building docs

- Checkout to `docs` first.
- Merge the master.
- Push -> Github action will run to build and deploy to github pages

```sh
> git checkout docs
```
