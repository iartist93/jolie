# Toggle Component

<toggle-1 />

## Details

The `Toggle` component allows users to **Switch** some boolean value on/off.

## Example

```html
<toggle v-model="isOpen" />
```

### Props

| Prop    | Type      | Default   |
| ------- | --------- | --------- |
| `value` | _Boolean_ | **false** |
